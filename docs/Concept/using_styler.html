Using DITAxPresso Styler

Styler allows you in customizing the output documents. You can change the attributes of the document with the Styler menu.

Current Version supports two types of Styler:
  • PDF Styler Customize XSLT for PDF
  • WebHelp Styler Customize XSLT for WebHelp
