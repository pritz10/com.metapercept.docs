Setting Plugin to Default Attributes

The user can change the Styler settings of the plugin to the default attributes with the steps below.
  1. Click Styler in the left-hand side navigation pane.
  2. Click PDF or WebHelp as required.
    Styler menu will be displayed in the workspace.
  3. On the top right-hand corner of the workspace, click Default.
  4. A pop-up message will ask, Restore Default Attributes. Click Confirm.
The settings are restored to default values.
