Creating a New Project

Pre-requisite:

  • JAVA runtime installed on the system in use.
  • An existing DITA Map.
This task guides you to create a new project in DITAxPresso.
  1. Open the DITAxPresso Software on the system.
  2. In the workspace, click New Project.
    A project window opens.
  3. Enter the following details :
    • Name of the project.
    • Description of the project. (optional)
    • Project-related keywords. (optional)
  4. Selecting DITA map folder
    1. Click folder icon against DITA map field.
    2. Browse to the existing DITA map folder and click.
    3. Click Select Folder.
  5. Selecting destination folder
    1. Click folder icon against Path field.
    2. Browse to the required output destination folder and click on the folder.
    3. Click Select Folder.
  6. Select the Initialize this project with READ.ME checkbox. (Optional)
  7. Click Create Project.
New Project is created.
